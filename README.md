# Day 63 — Recipe Finder App

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A recipe finder app that searches recipes by ingredients
or name using the MealDB free API.

## How to view
Open `index.html` in your browser.

## Features
- Search recipes by name or ingredient
- Filter by category and area
- Full recipe details with instructions
- Save favourite recipes to localStorage
- Random recipe generator
